# No-boxx-returns

Standalone "return with purpose" landing page for no-boxx — the reusable Velcro
return package. People scan the QR code on the pouch, land here, and learn how to
fill it, send it back, and support charity.

## Page

- **Route:** `/return-with-purpose/`
- **Files:** [`return-with-purpose/index.html`](return-with-purpose/index.html) + [`return-with-purpose/styles.css`](return-with-purpose/styles.css)

Pure static HTML/CSS — no build step, no dependencies. Open the HTML file directly
or serve the folder from any static host.

## Sections

1. Header — logo, nav (how it works / what you can send / faqs), "return with purpose" CTA
2. Hero — "fill it. send it. do good." + benefit cards + reusable pouch mockup with QR
3. How it works — 4-step process
4. What you can send — item grid + safety note
5. FAQs
6. Final CTA banner — "start your return"
